# fbartolic.github.io

Personal website of Fran Bartolić. Plain HTML and CSS, no build step — served
directly by GitHub Pages (`.nojekyll` disables the Jekyll pipeline).

- `index.html` — about
- `research/` — research overview and publications
- `cv/` — CV page; the PDF lives in `CV/` and is built from `CV/cv.tex` with `pdflatex`

To preview locally: `python3 -m http.server` from the repo root.
